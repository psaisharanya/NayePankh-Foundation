This folder is intended to hold the images used by the site.

To download the reference images from the original site, run the PowerShell script from the project root:

```powershell
./scripts/download_images.ps1
```

After running, you should see files like `hero.jpg`, `img1.jpg`, `impact1.jpg` etc.

If you prefer, manually place your optimized images in this folder and keep the same filenames.
